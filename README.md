# Ultra-Data-Burning-ROM!

Bringing back the feeling of burning CD-ROMs, but virtually.


- Upload files until you've got a full ROM
- Burn it as a disk image and save it to a decentralized durable storage network
- Browse a catalogue of ROMs published by other users
- Mount disk images and access the data

##

Run: `docker compose up -d`
Frontend will be available at localhost:8080
